Frequently Asked Questions

Everything you need to know about TBFusionAI

General Questions

What is TBFusionAI?

TBFusionAI is an AI-powered tuberculosis screening tool that analyzes cough sounds and clinical features to provide rapid TB screening results. It uses advanced machine learning models including Wav2Vec2 for audio feature extraction and an ensemble of gradient boosting models (CatBoost, XGBoost, LightGBM) for prediction.

Is TBFusionAI a diagnostic tool?

No. TBFusionAI is a screening tool, NOT a diagnostic device. It is designed to support healthcare professionals in identifying patients who may need further TB testing. Clinical diagnosis of TB requires confirmatory laboratory testing including sputum microscopy, culture, or molecular tests (e.g., GeneXpert).

How accurate is TBFusionAI?

TBFusionAI achieves a recall rate of 95% and an F1 score of 92% on validation data. The model is specifically optimized to minimize false negatives (missing TB cases), making it effective for screening purposes. However, like all AI systems, it has limitations and should be used as a supportive tool alongside clinical judgment.

Who developed TBFusionAI?

TBFusionAI was developed by AHFID AI Labs, leveraging the CODA TB dataset and state-of-the-art machine learning techniques. The project aims to make TB screening more accessible, especially in resource-limited settings.

Technical Questions

What technology powers TBFusionAI?

TBFusionAI uses several advanced technologies:

  • Wav2Vec2: Facebook's pre-trained model for audio feature extraction
  • Ensemble Models: Combination of CatBoost, XGBoost, and LightGBM
  • Cost-Sensitive Learning: Optimized to minimize false negatives
  • Audio Processing: Librosa and noisereduce for preprocessing
  • FastAPI: Modern web framework for the API

What audio formats are supported?

TBFusionAI supports WAV, MP3, and OGG audio formats. The audio should be:

  • Minimum duration: 1 second
  • Maximum file size:10MB
  • Recommended: 16kHz sample rate
  • Clear cough sound without excessive background noise

How long does analysis take?

Analysis typically takes 20-30 seconds, including:

  • Audio preprocessing and feature extraction: ~10-15 seconds
  • Model inference: ~5-10 seconds
  • Result generation: ~1-2 seconds

Is there an API available?

Yes! TBFusionAI provides a RESTful API for integration into other systems. Visit the API documentation for details on:

  • Authentication
  • Endpoint specifications
  • Request/response formats
  • Code examples in multiple languages

Usage Questions

What information do I need to provide?

For accurate screening, you need to provide:

  • Audio: Cough recording
  • Demographics: Age and sex
  • Symptoms: Cough duration, fever, weight loss, night sweats, hemoptysis
  • History: Prior TB history

How do I interpret the results?

Results include:

  • Prediction: "Probable TB" or "Unlikely TB"
  • Probability: 0-100% confidence in the prediction
  • Confidence Level: High, Medium, or Uncertain
  • Recommendation: Suggested next steps

Important: Always consult a healthcare professional regardless of the screening result.

What should I do if the result is "Probable TB"?

If the screening indicates "Probable TB", you should:

  1. Consult a healthcare professional immediately
  2. Undergo confirmatory TB testing (sputum test, chest X-ray, etc.)
  3. Follow medical advice regarding isolation and treatment
  4. Inform close contacts who may need screening

Remember: This is a screening tool, not a diagnosis. Only laboratory tests can confirm TB infection.

Can I use TBFusionAI for children?

TBFusionAI can process data for individuals of all ages (0-120 years). However, the model was trained primarily on adult data, so results for children should be interpreted with additional caution. Pediatric TB diagnosis often requires specialized expertise and additional considerations.

Privacy & Security

Is my data stored?

Audio files are processed in real-time and are NOT permanently stored on our servers. Clinical information is used only for the prediction and is not retained after the session ends. We prioritize user privacy and data security.

Is TBFusionAI HIPAA compliant?

The current version of TBFusionAI is a research tool and demonstration platform. For clinical deployment requiring HIPAA compliance, additional security measures and compliance certifications would need to be implemented. Contact us for enterprise deployment options.

Can I access my previous results?

Currently, results are not saved between sessions. We recommend printing or saving your results immediately after receiving them. Future versions may include user accounts with result history.

Support

I'm getting an error. What should I do?

Common issues and solutions:

  • File upload error: Ensure your audio file is under 10MB and in WAV/MP3/OGG format
  • Validation error: Check that all required fields are filled correctly
  • Timeout: Reload the page and try again with a shorter audio file
  • Connection error: Check your internet connection

If problems persist, please contact support.

How can I report a bug or suggest a feature?
